First release.

## What's in it

`ph` reads the Product Hunt public Atom feed at `https://www.producthunt.com/feed`.
No account, API key, or OAuth token is needed.

### Commands

| command | what it does |
|---|---|
| `ph today` | List today's top products from the feed |
| `ph feed` | Alias for `ph today` |

All commands support `-o table|json|jsonl|csv|tsv|url|raw`, `--fields`,
`--template`, `-n/--limit`, `--no-header`, `--quiet`.

### Output fields

Each product record has: `rank`, `name`, `tagline`, `author`, `published`, `url`.

## Install

```bash
go install github.com/tamnd/producthunt-cli/cmd/ph@v0.1.0
```

Or download a prebuilt binary from the [releases page](https://github.com/tamnd/producthunt-cli/releases/tag/v0.1.0).

### Docker

```bash
docker pull ghcr.io/tamnd/ph:0.1.0
docker run --rm ghcr.io/tamnd/ph:0.1.0 today -n 5
```
